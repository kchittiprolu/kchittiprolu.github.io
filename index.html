  • ShowWeatherNow Mobile App

    jQuery Mobile
    HTML5
    CSS3
    PhoneGap

    ShowWeatherNow App is a cross-platform hybrid app developed taking advantage of jQuery Mobile framework. This app has been converted to a native android app using Adobe's Phonegap. ShowWeatherNow app comes with features like current location, current temperature, hourly weather for next 24 hours, settings that include fahrenheit to celcius and vice versa. This app has a gear recommendation feature that recommends if you need a winter jacket, umbrella, shades or gloves based on the weather conditions for next few hours. Alternatively you can ask the app to show you recommended gears for next 2 to 12 hours.
